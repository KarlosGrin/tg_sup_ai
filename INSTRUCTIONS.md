## 🚀 Быстрый старт

1. **Отправьте боту файл** — .xlsx, .csv, .docx или .txt
2. **Бот проанализирует** структуру: листы, колонки, типы данных, количество строк
3. **Напишите команду** — что нужно сделать с данными
4. **Бот выполнит** задачу через AI и вернёт готовый файл

## 📊 Работа с Excel

### Что умеет бот с Excel:

| Действие | Пример команды |
|----------|----------------|
| **Чтение структуры** | Автоматически при загрузке |
| **Фильтрация строк** | *"Оставь только строки, где сумма > 1000"* |
| **Сортировка** | *"Отсортируй по дате от новых к старым"* |
| **Удаление дубликатов** | *"Удали дубликаты по колонке Артикул"* |
| **Добавление колонок** | *"Добавь колонку с процентом от общей суммы"* |
| **Математика** | *"Посчитай сумму продаж по месяцам"* |
| **Группировка** | *"Сгруппируй по региону и покажи средние продажи"* |
| **Объединение таблиц (VLOOKUP)** | *"Перенеси цены из второго файла в первый по артикулу"* |
| **Объединение листов** | *"Объедини все листы в один"* |
| **Разделение данных** | *"Раздели данные по месяцам на отдельные листы"* |
| **Форматирование** | *"Убери группировки и объединение ячеек"* |
| **Сводные таблицы** | *"Сделай сводную таблицу продаж по менеджерам"* |
| **Конвертация** | *"Сохрани как CSV"* / *"Сохрани как TXT"* / *"Сохрани как DOCX"* |

### Особенности:
- Бот видит все листы в книге
- Можно указать конкретный лист: *"На листе TDSheet посчитай сумму"*
- Первая строка автоматически считается заголовками колонок
- Точность данных сохраняется (даты, числа с плавающей точкой)

## 🔄 Конвертация между форматами

Одна из ключевых возможностей бота — конвертация данных между форматами.

| Из \ В | Excel (.xlsx) | Excel (.xls) | CSV | ODS | Word | TXT | XML |
|---------|-------------|--------------|-----|-----|------|-----|-----|
| **Excel (.xlsx)** | — | ✅ | ✅ | ✅ | ✅ | ✅ | ✅ |
| **Excel (.xls)** | ✅ | — | ✅ | ✅ | ✅ | ✅ | ✅ |
| **CSV** | ✅ | ✅ | — | ✅ | ✅ | ✅ | ✅ |
| **ODS** | ✅ | ✅ | ✅ | — | ✅ | ✅ | ✅ |
| **Word** | ✅ | ✅ | ✅ | ✅ | — | ✅ | ✅ |
| **TXT** | ✅ | ✅ | ✅ | ✅ | ✅ | — | ✅ |
| **XML** | ✅ | ✅ | ✅ | ✅ | ✅ | ✅ | — |

**Примеры:**
- *"Сохрани данные из Excel в Word документ"*
- *"Конвертируй CSV в Excel с форматированием"*
- *"Сохрани как TXT файл"*
- *"Сделай из этой таблицы Word документ с таблицами"*

## ⚠️ Ограничения

| Параметр | Значение |
|----------|----------|
| Макс. размер файла | 20 МБ |
| Запросов в минуту | 5 |
| Загрузок в час | 20 |
| Время выполнения кода | макс. 120 секунд |
| Кол-во листов в анализе | первые 3 листа |
| Сессия | хранится в памяти (сбрасывается при перезапуске) |

## 🔧 Технические детали

- **AI провайдер**: Gemini 3.1 Flash Lite (JSON mode)
- **Прокси**: SOCKS5 (для регионов с блокировкой Telegram)
- **Безопасность**: блокируются опасные операции (os.system, subprocess и т.д.)
- **Кэширование**: файлы хранятся в папках `downloads/` и `processed/`
- **Очистка**: автоматическая очистка файлов старше 24ч
